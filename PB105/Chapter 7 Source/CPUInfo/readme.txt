1. First copy CpuInfo.dll to the system directory, windows 9X is in the system directory of the windows directory, and windows 2000/XP is in the system32.
2. Click Run in the "Start" menu and enter:
regsvr32 C:\windows\system\cpuinfo.dll
or:
regsvr32 C:\windows\system32\cpuinfo.dll
Click OK to complete the registration of the dll.

The VBSource subdirectory is the source code in VB.
